# 产品利润计算功能使用说明

## 功能概述

产品利润计算功能允许您为产品设置报价，自动计算利润和利润率，并将结果保存到产品管理中。

## 数据库配置

**重要：** 首先需要联系数据库管理员执行以下SQL脚本：

```
apps/backend-mock/scripts/database-schema-updates.sql
```

该脚本会：
1. 在 `products` 表添加 `first_leg_shipping`（头程运费）、`profit`（利润）、`profit_rate`（利润率）字段
2. 创建 `product_profit_calculations` 表用于存储利润计算记录

## 功能说明

### 1. 产品管理增强

在产品管理页面，新增了以下功能：

- **头程运费列**：在国内运费和海外运费之间新增头程运费列
- **利润列**：在海外运费后显示利润信息（上行显示利润金额，下行显示利润率）
- **产品编辑**：添加产品或编辑产品时可以填写头程运费
- **批量导入**：Excel模板中包含头程运费字段

### 2. 产品利润计算

在"利润核算"模块下新增"产品利润计算"子菜单，功能包括：

#### 添加产品
- 点击"添加产品"按钮
- 从下拉列表选择产品
- 系统自动填充产品成本、税金、运费等信息

#### 计算利润
表格中包含以下字段：

**成本相关：**
- 产品成本
- 税金
- 国内运费
- 头程运费
- 总成本（自动计算 = 产品成本 + 税金 + 国内运费 + 头程运费）

**报价相关：**
- 报价/RMB（手动填写）
- 报价/JPY（自动换算，根据当前日元汇率）
- 运费补贴（手动填写，单位：人民币）

**利润相关：**
- 利润/RMB（自动计算 = 报价/RMB - 总成本 + 运费补贴）
- 利润率（自动计算 = 利润/RMB ÷ 报价/RMB × 100%）
- 无补贴利润率（自动计算 = (报价/RMB - 总成本) ÷ 报价/RMB × 100%）

#### 保存利润
- 点击"保存利润"按钮
- 系统将利润和利润率保存到产品管理
- 在产品管理列表中可以看到更新后的利润信息

## 计算公式

```
总成本 = 产品成本 + 税金 + 国内运费 + 头程运费

报价/JPY = 报价/RMB ÷ 日元汇率

利润/RMB = 报价/RMB - 总成本 + 运费补贴

利润率 = (利润/RMB ÷ 报价/RMB) × 100%

无补贴利润率 = ((报价/RMB - 总成本) ÷ 报价/RMB) × 100%
```

## 使用流程

1. **准备产品数据**
   - 在产品管理中添加产品
   - 确保填写完整的成本信息（产品成本、税金、国内运费、头程运费）

2. **计算利润**
   - 进入"利润核算" > "产品利润计算"
   - 添加需要计算利润的产品
   - 填写报价/RMB和运费补贴
   - 系统自动计算利润和利润率

3. **保存结果**
   - 点击"保存利润"按钮
   - 利润信息自动更新到产品管理

4. **查看利润**
   - 在产品管理列表中查看每个产品的利润和利润率

## 注意事项

1. 日元汇率目前使用固定值（1 JPY = 0.045 RMB），如需实时汇率请联系开发人员
2. 利润率默认显示有补贴利润率
3. 删除产品利润计算记录不会影响产品管理中已保存的利润数据
4. 建议定期更新产品的利润计算，确保数据准确性
